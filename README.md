# The johnson project
## Process all lakecontours within Finlands boundaries to detect certain shapes of great interest.


I trained a yolo v8 object detection model using sketches (30k images) from the internet. Then i used OSM and overpass api to get all lakes in Finland above certain area (2500m^2) threshold. Then i just detect. The CNN is overfitting a bit but definitely found me some gems.

### Links
I built this during the 6 week program [Sidequest](https://mysidequest.xyz/s1/projects)
